About

The cloud computing developer directory (c2d2) is a listing of tools and services from cloud computing developers for cloud computing developers. In order to be listed, the tool or service must be available in the cloud, that is, as a Web service or Web application accessible via a Web browser and/or offering an HTTP interface. The goal is to inform developers that write Web applications or deal with BigData in the cloud to quickly assess available solutions.

c2d2 was initiated by Michael Hausenblas and is in the public domain. The source data is managed via a Google Document, so if you want to contribute, request edit access, there.
